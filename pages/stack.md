## Install
```sh
npm install --save @datastructures-js/stack
```

## require
```js
const { Stack } = require('@datastructures-js/stack');
```

## import
```js
import { Stack } from '@datastructures-js/stack';
```

## JS
```js
const stack = new Stack([1,2,3,4]);
```

## TS
```js
const stack = new Stack<number>([1,2,3,4]);
```
